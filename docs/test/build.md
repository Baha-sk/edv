# EDV - Build

## Prerequisites (General)
- Go 1.13

## Prerequisites (for running tests and demos)
- Go 1.13
- Docker
- Docker-Compose
- Make

## Targets
```
# run all the project build targets
make all

# run license and linter checks
make checks

# run unit tests
make unit-test

# run bdd tests
make bdd-test
```
